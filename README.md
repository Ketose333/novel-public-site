# 이상기록 위키

`그저 깨어 있기를 바랄 뿐`과 `규칙을 판독하는 중입니다`의 공개용 위키형 작품 사이트입니다.

이 레포에는 웹에 노출해도 되는 공개 데이터와 Vite 앱만 포함합니다. 캐릭터 이미지, 내부 설정 원천, 연재 운영 문서는 포함하지 않습니다.

공개 데이터는 내부 문서를 완전히 분리한 홍보 문구가 아니라, 공개 등급에 맞춰 재편집한 위키 문서로 관리합니다.

## 개발

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

공개 데이터는 `src/data/site_data.json`을 읽습니다. 작품, 사건, 캐릭터, 용어 항목은 작업공간 내부 문서의 `site-work`, `site-incident`, `site-character`, `site-term` 공개 블록에서 생성하므로, 문장을 바꿀 때는 내부 문서를 먼저 수정한 뒤 동기화합니다.
